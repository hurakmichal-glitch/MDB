# MDB - Kodi Addons Repository

Kodi repository pre TV, Filmy a Seriály.

## Dostupné Addons

- **TV** - Addons pre TV stanice
- **Filmy** - Addons pre filmy
- **Seriály** - Addons pre TV seriály

## Inštalácia

1. Stiahnite si repozitár
2. V Kodi prejdite na: Nastavenia → Systém → Addons → Inštalovať zo ZIP súboru
3. Vyberte stiahnený repozitár

## Štruktúra

```
MDB/
├── repo/
│   └── plugin.video.mdb/
├── addons.xml
├── addons.xml.md5
└── README.md
```
