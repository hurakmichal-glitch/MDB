# ZIP Addons Directory

Tento adresár obsahuje ZIP balíčky jednotlivých addons.

## Štruktúra

- `plugin.video.mdb.tv/` - TV addon ZIP súbory
- `plugin.video.mdb.filmy/` - Filmy addon ZIP súbory
- `plugin.video.mdb.serialy/` - Seriály addon ZIP súbory

## Ako pridať ZIP súbory

1. Vytvorte ZIP balíček vášho addonu
2. Pomenujte ho podľa formátu: `plugin.video.mdb.xxx-verzia.zip`
3. Vložte ho do zodpovedajúceho adresára

Príklad: `plugin.video.mdb.tv-1.0.0.zip`
